# UrbanKick Theme

Custom Fynd Commerce theme built for the Nuviratek Hackathon.

## Features

- Custom Homepage
- Product Listing Page (PLP)
- Product Detail Page (PDP)
- Cart Page
- Checkout Styling
- Account Pages
- Terms & Conditions Page
- Return Policy Page
- FAQ Page

## Technology Used

- Fynd Theme Framework
- React
- JavaScript
- CSS
